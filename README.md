# Sales-Data-Analysis
## 📌 Project Overview
This project analyzes sales data using Python and Pandas. It reads sales data from an Excel file, performs analysis, and calculates total sales revenue.

## 🚀 Features
- Read Excel data using Pandas
- Display dataset preview
- Show dataset information
- Generate statistical summary
- Calculate Total Sales
- Calculate Total Revenue

## 🛠 Technologies Used
- Python
- Pandas
- Excel (.xlsx)

## 📂 Project Files
- salesdataanalysis.py
- Book1.xlsx

## ▶️ How to Run
1. Install Python
2. Install Pandas (`pip install pandas openpyxl`)
3. Keep `Book1.xlsx` in the same folder as `salesdataanalysis.py`
4. Run:
   ```
   python salesdataanalysis.py
   ```

## 📊 Output
- Dataset Preview
- Dataset Information
- Statistical Summary
- Total Sales Calculation
- Total Revenue
